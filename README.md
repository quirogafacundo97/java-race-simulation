# Java Race Simulation

Proyecto realizado en Java aplicando Programación Orientada a Objetos mediante la simulación de una carrera de autos.

## Características

- Simulación de movimiento de vehículos
- Interacción entre múltiples objetos
- Manejo de estados
- Relación entre clases
- Uso arreglos según implementación

## Conceptos trabajados

- Clases y objetos
- Herencia
- Composición
- Encapsulación
- Relaciones UML
- Simulación de comportamiento

## Posibles entidades utilizadas

- Auto
- Motor
- Tanque
- Carrera
- Conductor

## Tecnologías utilizadas

- Java
- Eclipse IDE
